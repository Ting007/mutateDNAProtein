The code is to generate the files of shared telephone number.
To run the program under python 3.0, type the command in your terminal:
python phoneName.py phoneDir.txt output.txt

-source code in phoneName.py.
-phoneDir.txt is the input file, 2 column tab-delimited file, name + 'tab' + phone. Or you can replace it with your input file .txt.
-output.txt is the output file, 2 column tab-delimited file, phone number + 'tab' + names separated by comma. Or you can name it as you like.

To unitTest the program under Python 3.0 unittest, run the command in your terminal:
python phoneNameTest.py

Make sure you are using python 3.0.
